DloucFlare

Your server is running under a dynamic IP? Want something to modify DNS record to the real-time IP address automatically? Maybe DloucFlare can help you. In fact, Zanarberry is using it.
First of all, you should have a CloudFlare account.
Fork me on GitHub.

XTO

Just an API for Node.js to query each express. You can easily install it via npm
Because many Chinese express companies are called [*]to (eg: sto, zto, gto, etc.), I call this project XTO.
Thread at CNodeJS and you can also fork me on GitHub.
